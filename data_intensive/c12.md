# Chapter 12: The Future of Data Systems

## 2. Data Integration

### 2.1 Vấn đề

- Một hệ thống thực tế gồm:
    - OLTP database
    - Data warehouse
    - Stream system
    - Search index

→ Data bị phân mảnh

---

### 2.2 Giải pháp:

- **Data integration**

---

### 2.3 Approaches

#### a. ETL (Extract – Transform – Load)

- Batch processing

---

#### b. Event-based integration

- Stream (Kafka)

---

### 2.4 Two philosophies

#### Data duplication

- Copy data giữa systems

✔ Decoupling  
❌ Sync khó

---

#### Single source of truth

- Một nguồn duy nhất

✔ Consistency  
❌ Hard to scale

---

## 3. Unbundling Databases

### 3.1 Truyền thống:

- Database = tất cả trong một

---

### 3.2 Hiện đại:

- Tách ra:
    - Storage
    - Index

    - Processing

### 3.3 Ví dụ:

- Kafka → log
- Elasticsearch → search
- OLTP DB → transaction

## 10. Key Insights

- Không có "one-size-fits-all"
- Modern system = tập hợp nhiều specialized tools
- Log (event stream) là trung tâm
- Derived data là core concept
- Batch & Stream đang hội tụ
